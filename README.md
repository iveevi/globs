This repository contains GLB (glTF binary) versions of various scenes,
converted from their original sources. These are arranged into the tables
below by license.

We also provide a reference scene viewer in viewer/ using Slang, SlangPy and
`pygltflib`. Run it with the following:

```
uv run viewer/main.py <file>
```
